# CrankScribe

A voice-powered note-taking app for Playdate that streams audio to a cloud transcription service and displays results on the device's 1-bit screen.

## Overview

CrankScribe turns your Playdate into a dedicated voice recorder that captures meetings, thoughts, and ideasâ€”then transcribes and processes them via streaming Whisper. The crank provides unique interactions for scrubbing through recordings and transcripts.

## Requirements

- Playdate SDK 3.0+
- A WhisperFlow-compatible transcription server
- WiFi connection for streaming

## Development

### Setup

1. Install the [Playdate SDK](https://play.date/dev/)
2. Clone this repository
3. Build with `pdc source CrankScribe.pdx`
4. Run in the Playdate Simulator or sideload to device
